# Configure Route Tables in a Virtual Network

## About
In this challenge, you will ensure that all traffic between the front-end and back-end subnets of the virtual network are routed through a firewall virtual appliance. First, you will create the route tables and routes, and then you will then associate them to the subnets. Next, you will enable IP forwarding on the virtual appliance. Finally, you will verify that traffic is being routed through the virtual appliance.
