# Configure an Azure Lock 

## About

In this challenge, you will protect Azure resources from modification and deletion. First, you will apply locks to an Azure resource, and then you will test the locks. Next, you will apply a lock to a resource group. Finally, you will test the inheritance of the lock. Note: Once you begin a challenge you will not be able to pause, save, or return to your progress. Please ensure you have set aside enough time to complete the challenge before you start.
