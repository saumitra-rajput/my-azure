## About
In this challenge, you will create multiple virtual networks and peering connections for secure bidirectional communication. First, you will create an Azure virtual network by using the Azure portal. Next, you will create an Azure virtual network by using Azure Cloud Shell. Finally, you will configure virtual network peering connections for secure bidirectional communication.
