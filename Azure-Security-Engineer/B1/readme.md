# Configure Virtual Network Connectivity by Using Peering 

## About
In this challenge, you will create multiple virtual networks and peering connections for secure bidirectional communication. First, you will create an Azure virtual network by using the Azure portal. Next, you will create an Azure virtual network by using Azure Cloud Shell. Finally, you will configure virtual network peering connections for secure bidirectional communication. Note: Once you begin a challenge you will not be able to pause, save, or return to your progress. Please ensure you have set aside enough time to complete the challenge before you start.
