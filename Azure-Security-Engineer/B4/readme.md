
# Configure Application Insights and App Service Logs for a Web App

## About
In this challenge, you will create and deploy an Azure web app that includes Application Insights and App Service logs. First, you will create a web app, and then you will deploy the source code for the web app. Next, you will enable Application Insights for monitoring. Finally, you will enable App Service logs for auditing and debugging.
