# Enable Database Auditing with Azure SQL Database

## About
In this challenge, you will create a new Azure SQL database that uses database auditing. First, you will create an Azure SQL database. Next, you will enable database auditing by using a storage account for the audit log. Finally, you will review the audit log. Note: Once you begin a challenge you will not be able to pause, save, or return to your progress. Please ensure you have set aside enough time to complete the challenge before you start.
